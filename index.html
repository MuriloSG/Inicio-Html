Estrutura do HTML:

A estrutura do HTML é bem simples e de fácil entendimento, inicialmente define o tipo de documento, (doctype) serve para dá instruções para o navegador ler o HTML, em seguida vem a tag filha (head), nela tem informações que não é mostrada para o usuário, por fim o (body), que é o corpo do web site, ou seja, tudo aquilo que será visto pelo o usuário.

Tags:

Tags são usadas para informar ao navegador a estrutura do site. Ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página, tags aprendidas no curso dio: h1, p, mark>, small, i, u, strong, ol, ul, li, a, hr, sub, sup, blockquote, font, del, abbr.

Lista ordenada:

A marcação da tag de lista ordenada é "ol", essa tag é utilizada para mostrar uma lista de forma crescente, com numeros, algarismo romano, letras etc. Utilizada em conjunto com "li", é usado para representar um item que faz parte de uma lista.

Lista não ordenada:

A marcação da tag de lista não ordenada é "ul", essa tag é utilizada para mostrar uma lista de itens, sem ordem crescente, aparece apenas o marcador definido pelo dev. Utilizada em conjunto com "li", é usado para representar um item que faz parte de uma lista.

Links:

A marcação da tag de link é "a", com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada "a" precisará indicar o destino do link.
